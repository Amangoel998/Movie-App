# Movie-App
This is simple Movie Application that is intended to add and view movies and their rating.

```
//Linux
./mvnw clean install
./mvnw spring-boot:run    

//Windows
./mvnw.cmd clean install
./mvnw.cmd spring-boot-run

//To run Angular
cd ./frontend
npm start
```
